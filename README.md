## üì¶ Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/naeemqayyum/payroll-system.git
   cd payroll-system
   ```

2. **Build the app**
   ```bash
   mvn clean install
   ```

